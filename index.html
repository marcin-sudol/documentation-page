General information

SQL stands for Structured Query Language. It's a language used in relational databases. SQL commands can be divided into four groups:

  • DQL (Data Query Language) - for querying data
  • DML (Data Manipulation Language) - for modifying data
  • DDL (Data Definition Language) - for creating and modifying database objects
  • DCL (Data Control Language) - for controling access to data
DQL statements

There is only one statement used for querying the data:

  • SELECT

This statement can be complex and consist of many clauses.

Sometimes SELECT statement is considered as a part of DML.

DML statements

DML consists of three statements:

  • INSERT - for inserting data into table
  • UPDATE - for updating data in table
  • DELETE - for deleting data from table
DDL statements

Statements included in DDL:

  • CREATE - for creating objects like databases, tables, indexes and other
  • DROP - for removing objects
  • ALTER - for modifying existing objects
  • TRUNCATE - for removing all data from table
DCL statements

Statements included in DCL:

  • GRAND - for granting permission to perform specific actions
  • REVOKE - for removing permission to perform specific actions
SELECT query structure

SELECT statement consists of 6 clauses:

SELECT ...
FROM ...
WHERE ...
GROUP BY ...
HAVING ...
ORDER BY ...

Only SELECT and FROM clauses are mandatory.

Examples of SELECT statement:

SELECT FirstName, LastName
FROM Customers SELECT ProductName
FROM Products
WHERE ProductCategoryID = 2
ORDER BY ProductName
